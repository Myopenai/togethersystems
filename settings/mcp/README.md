## Übersicht

Dieses Verzeichnis enthält die MCP-Integration für TogetherSystems. MCP (Model Context Protocol) ermöglicht die Kommunikation zwischen AI-Agents und externen Tools/Services.

- `HEADING-ANCHOR-PROJECT.md` - Hauptprojekt-Dokumentation

---

## Integration

Die MCP-Integration ist Teil der Industrial Fabrication Routine und wird automatisch aktiviert:

- **Pre-Workflow:** Alle MCP-Tools werden aktiviert
- **During-Workflow:** MCP-Tools werden für Code-Verification verwendet
- **Post-Workflow:** MCP-Tools werden für Tests und Validierung verwendet

---

## Verfügbare MCP-Tools

1. **Playwright** - Browser-Tests und UI-Validierung
2. **Codebase-Search** - Architektur-Analyse und Konsistenz-Prüfung
3. **File-Operations** - Atomare, geprüfte Schreiboperationen
4. **Terminal** - Tests, Linter, Builds
5. **Web-Search** - Standards, Spezifikationen, Referenzen

## BEISPIELE

### Beispiel 1: User-ID Generation

**Formel:**
```
userId = Base62(Entropy(128 bits))
```

**JavaScript-Implementierung:**
```javascript
function generateUserId() {
  const entropy = crypto.getRandomValues(new Uint8Array(16));
  return base62(entropy);
}
```

**Python-Implementierung:**
```python
def generate_user_id():
    entropy = secrets.token_bytes(16)
    return base62(entropy)
```

### Beispiel 2: Voucher Validation

**Formel:**
```
valid(voucher) = (
  voucher.status ≠ 'expired' ∧
  voucher.status ≠ 'cancelled' ∧
  now ≥ voucher.valid_from ∧
  now ≤ voucher.valid_until
)
```

**JavaScript-Implementierung:**
```javascript
function isValidVoucher(voucher) {
  const now = Date.now();
  return voucher.status !== 'expired' &&
         voucher.status !== 'cancelled' &&
         now >= voucher.valid_from &&
         now <= voucher.valid_until;
}
```

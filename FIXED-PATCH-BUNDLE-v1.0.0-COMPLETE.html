Die Applikation ist als statisches Portal ausgelegt.

4.1 Lokal

1. ZIP-Bundle entpacken. 2. In das Verzeichnis `v1.0.0-PRODUCTION-20251124-222131/` wechseln. 3. Einen einfachen HTTP-Server starten, z. B.: ```bash python -m http.server 9323 ``` 4. Browser öffnen und aufrufen: ```text http://localhost:9323/v1.0.0-PRODUCTION-20251124-222131/ ``` 5.
